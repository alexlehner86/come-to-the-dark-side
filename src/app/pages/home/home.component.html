Introduction

What does it feel like to be blind? How can you navigate a website and read the content? Globally, at least 2.2 billion people have a vision impairment or blindness. If you're not one of them, now is your chance to put yourselves in their shoes!

This project wants to enable all users to experience a website without visual input, relying only on the feedback a screen reader provides. Turn everything black via the "Go Dark" button and experience the website as a blind user would.

Instructions

Follow these steps:

  1. Install and activate the screen reader on your device: e.g. NVDA on Windows, VoiceOver on Apple devices, and TalkBack on Android.
  2. Familiarize yourself with the operation of the screen reader. For example, NVDA offers many useful keyboard shortcuts.
  3. Now push the button labelled "Go Dark" in the right upper corner. This causes everything to be painted black and all highlighting on focus or hover to be switched off. The only way to interact with the website will be via your screen reader together with a keyboard on desktop devices, or touch gestures on mobile devices.
  4. Have fun!
